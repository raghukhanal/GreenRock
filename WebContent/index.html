GreenRock

This is GreenRock, our project extracts data from the Aladdin API, and processes the financial data to help consumers make better financial decisions. We also incorporated an interactive and extremely accurate game using advanced ML models to predict future portfolio changes. Using the data, we also made a fun fact page, demonstrating the growth of your portfolio if you would have invested 10,000 dollars five years ago.

Our Project Overview

After extracting the data from the Aladdin Portfolio Analysis API. We realized that the data only had past data on the stocks of these companies. We generated a where you will be dropped in January 2019, where you can invest any amount of money to a variety of stocks, and our Machine Learning model can accurately calculate the worth of your portfolio on any certain day, all the way up to last week which is where the provided Aladdin API data ends. We also made a information slide showing the effects of investing in stock, this increases financial literacy because people can see the growth of portfolios over a time span encouraging people to invest reasonably. FINALLY, Using advanced neural networks, we were able to reasonably determine if a stock is overpriced or under-priced.
